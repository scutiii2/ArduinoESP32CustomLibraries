# Arduino & ESP32 Custom Libraries 🚀

Welcome to my collection of custom libraries for **Arduino** and **ESP32** development.  
This repository is designed to make embedded programming easier, faster, and more modular by providing reusable components for common tasks.

---

## ✨ Features
- Lightweight and optimized for microcontrollers
- Compatible with **Arduino IDE** only
- Well-documented examples for quick integration
- Modular design for easy customization

---

## 📚 Libraries Included
Here’s the list of libraries currently available in this repository:

- **CommandLib** – Lightweight command parser and controller for Arduino.
- **IRRemote** – Simple Infrared Remote Controller.
- **RTCLite** – Lightweight non-blocking DS3231 RTC wrapper.
- **RTOSLite** – Lightweight RTOS-like cooperative scheduler for Arduino and ESP32.
- **ToneLite** – Non-blocking tone player for Arduino and ESP32.

---
